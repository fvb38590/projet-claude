# Vitrine — Automatisation pour PME

Fichiers: `index.html`, `styles/styles.css`

Prévisualiser localement:

Option 1 — Python (port 8000):

```bash
cd d:/projet-claude
python -m http.server 8000
# puis ouvrir http://localhost:8000/landing-pme-automation/
```

Option 2 — VS Code: installer l'extension Live Server et ouvrir `landing-pme-automation/index.html`.

Prochaines étapes possibles:
- Personnaliser les textes et le contact email
- Ajouter logos et images
- Préparer un commit Git et créer une PR
